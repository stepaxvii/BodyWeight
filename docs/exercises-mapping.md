## Категории (по типу нагрузки)

| Slug | Name | Name_RU | Описание |
|------|------|---------|----------|
| `strength` | Strength | Силовые | Преодоление сопротивления |
| `cardio` | Cardio | Кардио | Повышение пульса, выносливость |
| `static` | Static | Статика | Удержание позиции |
| `dynamic-stretch` | Dynamic Stretch | Динамическая растяжка | Растяжка в движении |
| `static-stretch` | Static Stretch | Статическая растяжка | Удержание позы растяжки |
